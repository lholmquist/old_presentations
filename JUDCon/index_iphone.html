    AeroGear-Android

    • Android 2.1+ support
    • Example:
    // create an empty pipeline:
Pipeline pipeline =
  new Pipeline("http://todo-aerogear.rhcloud.com/todo-server/");

// Add a connection/pipe object:
Pipe tasks = pipeline.pipe(Task.class, pipeConfigBean);

// issue HTTP GET: 
tasks.read(new Callback<List<Task>>() {
  @Override
  public void onSuccess(List<Task> data) {
    // work with the data....
  }
  @Override
  public void onFailure(Exception e) {}

};
