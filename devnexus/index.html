AeroGear Security

  • Authentication

@Stateless
public class Login {

    @Inject
    private AuthenticationManager authenticationManager;

    public AeroGearUser login(AeroGearUser user) {

        authenticationManager.login(user);

        return user;
    }

    public void logout() {
        authenticationManager.logout();
    }
}

AeroGear Security

  • Registration

@Stateless
public class Register {

    //For example purposes only
    public static final String DEFAULT_ROLE = "admin";

    @Inject
    private IdentityManagement configuration;

    public AeroGearUser register(AeroGearUser user) {
        configuration.create(user);
        configuration.grant(DEFAULT_ROLE).to(user);
        return user;
    }
}

What is an authenticator?

  • Collection of Authentication Modules
 

// Javascript
var auth = AeroGear.Auth();
                    

// iOS
AGAuthenticator* auth = [AGAuthenticator authenticator];
                    

// Android
Authenticator auth = new DefaultAuthenticator();

What is an auth module?

  • Authentication and Enrollment API
 

// Javascript
auth.add(
    {
        name: "myAuth",
        settings: {
            agAuth: true,
            baseURL: "/todo-server/"
        }
    }
);

auth.modules['myAuth'].login( ... );

auth.modules['myAuth'].enroll( ... );

auth.modules['myAuth'].logout( ... );

What is an auth module?


// iOS
_restAuthModule = [authenticator auth:^(id config) {
            [config name:@"restAuthMod"];
            [config baseURL:projectsURL];
        }];

[_restAuthModule login ... ]

What is an auth module?


// Android
Authenticationmodule authMod =
        Authenticator.auth(AuthType.REST, baseURL).add();

authModule.login( ... );

What is a datamanager?

  • Collection of Data Objects(Stores)
 

// Javascript
var dataManager = AeroGear.DataManager();
                    

// iOS
AGDataManager* dm = [AGDataManager manager];
                    

// Android
DataManager dataManager = new DataManager();

What is a store?

  • client side storage API
  • Read/Save/Remove/Filter
  • Sync Manager( coming soon )
 

// Javascript
datamanager.add( "projectStore" );

var projectStore = datamanger[ "projectStore" ];

projectStore.read( ... );

datamanager.add({
    name: "sessionStore",
    type: "SessionLocal"
});

var sessionstore = datamanager.stores.sessionStore;

sessionstore.read( ... );

What is a store?


// iOS
id myStore = [dm store:^(id config) {
    [config setName:@"tasks"];

[myStore read: ... ];

What is a store?


// Android

myStore.read( ... );

What is a pipeline?

  • Collection of Server Connections( Pipes )
 

// Javascript
var pipeline = AeroGear.Pipeline();
                    

// iOS
AGPipeline* pipeline = [AGPipeline pipeline:projectsURL];
                    

// Android
pipeline = new Pipeline(baseURL);

What is a pipe?

  • Server Connection
  • Read/Save/Remove
  • Async Callbacks
 

// Javascript
var simplePipe = pipeline.add( "simplePipe" ).pipes.simplePipe;

var myPipe = pipeline.add({
    name: "myPipe",
    config: {
        baseURL: "customBaseURL"
        endPoint: "customEndpoint"
    }
}).pipes.myPipe;

myPipe.read( ... );

What is a pipe?


// iOS
_myPipe = [pipeline pipe:^(id config) {
                [config name:@"myPipe"];
                [config endpoint:@"customEndpoint"];
            }];

[_myPipe read: ... ];

What is a pipe?


// Android
private Pipe myPipe;

pipeConfig.setName("myPipe");
pipeConfig.setEndpoint("customEndpoint");

pipeline.pipe(POJOReturn.class, pipeConfig);

myPipe = pipeline.get("myPipe");

myPipe.read( ... )
